CSS is not what you think

CSS is used for web pages, for sure.

CSS is (also) designed for paged media

Since CSS2 in 1998 (20th century, were you born yet?), it includes a "paged media" chapter.

What you could do

In CSS2.1, standard since 2011, you can:

  • force page breaks,
  • avoid page breaks,
  • define how many widow and orphan lines are allowed,
  • select first, left, right pages,
  • add text and page numbers in margins.
